# CONNECT GREEN

*A Smart Web Platform for Sustainable Tourism*

This project is a smart, web-based platform designed to unify the fragmented landscape of sustainable tourism. The platform connects eco-conscious tourists, verified green businesses, and protected natural sites.

## Tech Stack
- Frontend: React.js (Vite), Tailwind CSS, React Router, Recharts, Framer Motion
- Backend: Node.js, Express.js, MongoDB (Mongoose)

## Available Scripts

In the project directory, you can run:

### `cd server && npm run dev`
Runs the backend API server. Open [http://localhost:5000](http://localhost:5000) to view it in your browser.

### `cd client && npm run dev`
Runs the frontend React app. Open [http://localhost:5173](http://localhost:5173) to view it in your browser.
